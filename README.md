## Requirements

 * MODX Evolution 1.0.5 or greater.

## Installation

 * Upload directory "EvoRevoTheme" to "/manager/media/style/"
 * Login your MODX manager and go to Tools > Configuration > Interface & Features > Manager Theme. 
 * Choose "EvoRevoTheme" and save.
